# Orixis Digital Future

This is the official website for Orixis, a cutting-edge technology company specializing in AI/ML solutions, custom web development, mobile apps, and intelligent automation.

## Project info

This project is built with Vite, React, TypeScript, and Tailwind CSS.

## How to run locally

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd orixis-digital-future

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Lucide React Icons
